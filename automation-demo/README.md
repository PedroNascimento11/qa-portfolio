# Playwright Automation Demo — Demoblaze E-commerce

This test validates a critical purchase flow:
- Open product page
- Add item to cart
- Navigate to cart
- Assert product name and price match

## Run locally
npm install
npx playwright test

## Tech Stack
- JavaScript
- Playwright Test Runner

## Why this test matters
It covers a key business risk in e-commerce applications:
incorrect pricing / product mismatch in checkout.
